## 场地预定结算系统功能简介
>该系统主要实现：
>               根据小明每次收到的场地预定情况，计算当前预定条件下的各项收支情况，并统对一段时间内的总收益进行统计，方便小明直观了解组织活动的收支情况。

@(代码学习)
**运行环境**：
- **JDK 1.8**
- **Junit  4.12**
		
## 代码结构
本程序主要分为三个部分，分别是`Resources、Sources和Test`，其中Resources保存项目相关的配置信息，Source为主要的实现代码，Test为相应的单元测试用例。
